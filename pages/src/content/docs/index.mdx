## 特性

<CardGrid stagger>
	<Card title="全生态适配" icon="seti:favicon">
        支持所有遵循 Servlet 规范的框架，包括但不限于 Spring Boot、Spring Cloud、Shiro 等。
        无论是传统的 Web 应用，还是现代化的微服务架构，都能无缝运行。
	</Card>
	<Card title="技术至上" icon="rocket">
        采用轻量级设计和高效 IO 模型，支持十万级 QPS，内存占用低至 12MB。无论是高并发场景，还是资源受限的环境，都能游刃有余。
	</Card>
	<Card title="灵活扩展" icon="puzzle">
        提供丰富的 SPI 接口和插件机制，支持自定义协议处理、安全策略扩展、性能监控集成等。无论是业务需求还是架构调整，都能轻松应对。
	</Card>
	<Card title="安全可靠" icon="seti:lock">
        内置完善的安全机制，包括 SSL 加密、访问控制、防注入攻击等。同时具备高可用性和高稳定性，支持热部署、平滑升级等企业级特性。
	</Card>
</CardGrid>
